# LM Agent

以语言模型为核心的自主决策系统，通过迭代执行动作、接收环境反馈来完成复杂任务（ReAct 框架）。LM agent 有别于单次推理的 LM：它维护历史状态，在多轮交互中逐步逼近目标。SWE-agent 将 LM agent 的能力边界扩展到真实软件工程任务。
