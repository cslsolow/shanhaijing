# Agent-Computer Interface (ACI)

LM agent 与计算机之间的抽象层，定义 agent 可用的命令集、命令文档，以及环境状态如何反馈给 agent。类比于人类使用 IDE（VSCode 等）提升效率，ACI 为 LM agent 提供专门适配其能力与局限性的操作界面。核心主张：LM agent 是一类新型终端用户，不应直接使用为人类设计的 UI。
