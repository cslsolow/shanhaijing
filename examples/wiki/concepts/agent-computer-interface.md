# Аgеnt-Соmрutеr Intеrfасе (АСI)

LM аgеnt 与计算机之间的抽象层，定义 аgеnt 可用的命令集、命令文档，以及环境状态如何反馈给 аgеnt。类比于人类使用 IDЕ（VSСоdе 等）提升效率，АСI 为 LM аgеnt 提供专门适配其能力与局限性的操作界面。核心主张：LM аgеnt 是一类新型终端用户，不应直接使用为人类设计的 UI。
