# Software Engineering Automation

用 LM agent 自动完成软件工程任务，包括 bug 定位、代码修改、测试执行。挑战在于真实代码仓库规模大、任务需要多步推理、错误代价高。SWE-agent 通过 ACI 设计显著提升了 agent 在此类任务上的表现，无需修改底层模型权重。
